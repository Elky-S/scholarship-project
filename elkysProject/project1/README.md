# Custom Calendar Project 📅

אפליקציית לוח שנה אישית ודינמית שנבנתה ב-**React**. הפרויקט מאפשר למשתמשים לנהל את הזמן שלהם בצורה חכמה על ידי יצירת לוחות שנה מותאמים אישית והוספת אירועים בתאריכים נבחרים.

## ✨ תכונות עיקריות (Features)
* **יצירת לוחות מרובים:** אפשרות ליצור לוח שנה תחת שם ספציפי ולנהל מספר לוחות במקביל.
* **תצוגה דינמית:** חישוב אוטומטי של ימי החודש והשבוע.
* **ניהול אירועים:** הוספת אירועים (כולל אימוג'י ותיאור) לכל יום בלוח (עד 3 אירועים ליום).
* **שמירה מקומית (Persistence):** שימוש ב-`LocalStorage` לשמירת הנתונים בדפדפן, כך שהמידע נשמר גם לאחר רענון הדף.
* **עיצוב מותאם:** אפשרות לשינוי צבעי הלוח, הרקע והטקסט בהתאמה אישית.

## 🛠️ טכנולוגיות (Tech Stack)
* **Frontend:** React.js (Hooks, Refs)
* **Styling:** CSS3 (Flexbox, Grid)
* **Logic:** JavaScript (ES6+)
* **Alerts:** SweetAlert2 לממשק הודעות ידידותי.

## 🏗️ מבנה הפרויקט וצורת העבודה
הפרויקט בנוי במבנה מודולרי, תוך הפרדה ברורה בין הלוגיקה לתצוגה:

* **ניהול סטייט (State Management):** שימוש ב-`useState` לניהול האירועים והתאריכים, מה שמאפשר עדכון מיידי של התצוגה (Reactive UI).
* **רכיבים מופרדים (Component-Based):** חלוקה לרכיבים קטנים (כמו `AddEvent` ו-`CreateCalander`) לתחזוקה קלה ושימוש חוזר.
* **עיצוב דינמי:** שימוש ב-CSS גמיש המתאים את עצמו לתוכן המשתנה בלוח.

## 🚀 פונקציונליות טכנית
1.  **חישוב תאריכים:** שימוש באובייקט `Date` המובנה של JS לחישוב מדויק של פריסת הימים.
2.  **ממשק אירועים:** ניהול מערך אובייקטים מורכב הכולל חודשים, ימים ואירועים, והצגתם במיקום המדויק.
3.  **מבנה תיקיות עיקרי:**
    * `src/calander`: מכיל את רכיב הליבה של הלוח ואת קבצי העיצוב הייעודיים.
    * `src/App.js`: נקודת הכניסה המרכזית המאגדת את כל הרכיבים והלוגיקה הראשית.

## 💻 איך להריץ את הפרויקט מקומית?

1. בצעו Clone למאגר:
   ```bash
   git clone [https://github.com/Elky-S/custom-calendar-frontend.git](https://github.com/Elky-S/custom-calendar-frontend.git)

2.היכנסו לתיקיית הפרויקט:
```
cd custom-calendar-frontend
```
3.התקינו את התלויות:
```
npm install
```
4.הריצו את הפרויקט:
```
npm start
```
